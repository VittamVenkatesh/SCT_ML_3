# SCT_ML_3 - Cat and Dog Image Classification using SVM

##  Task Objective
Build an image classification model using Support Vector Machine (SVM) to classify images as Cats or Dogs.

##  Tools and Libraries Used
- Python
- Google Colab
- NumPy
- OpenCV (cv2)
- Matplotlib
- Scikit-learn

##  Dataset
Dataset: Cats and Dogs Image Dataset from Kaggle.

Classes:
- Cat 
- Dog 

##  Steps Performed
1. Uploaded and extracted the dataset in Google Colab.
2. Loaded images using OpenCV.
3. Resized images to 64 × 64 pixels.
4. Converted images into numerical arrays.
5. Split the dataset into training and testing sets.
6. Trained an SVM (Support Vector Machine) model.
7. Evaluated the model using accuracy score and confusion matrix.

##  Model Used
- Support Vector Machine (SVM)
- Kernel: Linear

##  Results
- Successfully classified cat and dog images.
- Generated prediction results and confusion matrix.
